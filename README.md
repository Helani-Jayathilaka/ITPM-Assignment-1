# IT3040-ITPM: Assignment 1 - Transliteration Testing

This repository contains the automated test script for evaluating the Singlish-to-Sinhala chat translator application using Playwright.

## Prerequisites
* Python 3.11 or 3.12 installed on your system.

## Installation Instructions
If the standard `pip` command is not recognized as an internal or external command on Windows, use the Python launcher (`py`) to install the dependencies.

1. Clone this repository to your local machine, or download the ZIP file and extract it.
2. Open your Command Prompt and navigate to the extracted folder.
3. Install the required dependencies by running:
   `py -m pip install playwright openpyxl`
   `py -m playwright install`

## How to Run the Tests
1. Ensure the `test_automation.py` script and the `Assignment 1 - Test cases.xlsx` file are in the same directory.
2. Run the following command in the terminal to execute the automation script:
   `py test_automation.py`
